# 個人コントリビューターライセンス契約（v1.0, OpenAI）

_Apache Software Foundation Individual CLA v 2.2 に基づく。_

Pull Request 上で **"I have read the CLA Document and I hereby sign the CLA"**
とコメントすることにより、**あなた（「Contributor」）は、以下の条件に同意するものとします。**この同意は、
過去および将来に **OpenAI Codex CLI project（「Project」）** に対して提出される
「Contribution」に適用されます。

---

## 1. 定義

- **「Contribution」** – Project に提出される、著作者によるオリジナルの成果物
  （コード、ドキュメント、デザインなど）を意味します。
- **「You」 / 「Your」** – 上記の承諾コメントを投稿する個人（または法人）を意味します。

## 2. 著作権ライセンス

あなたは、**OpenAI, Inc.** および Project によって配布されるソフトウェアのすべての受領者に対し、
あなたの Contribution およびその派生物を複製し、派生物を作成し、公に表示し、公に実演し、
再許諾し、配布するための、永続的、全世界的、非独占的、ロイヤリティフリーで、
取消不能なライセンスを付与します。

## 3. 特許ライセンス

あなたは、**OpenAI, Inc.** および Project のすべての受領者に対し、あなたの Contribution を
単体で、または Project と組み合わせて、製造し、製造させ、使用し、販売し、販売の申出を行い、
輸入し、その他譲渡するための、永続的、全世界的、非独占的、ロイヤリティフリーで、
（以下に定める場合を除き）取消不能な特許ライセンスを付与します。

いずれかの者が、Project または Contribution が特許を侵害していると主張して特許訴訟を提起した場合、
本 CLA に基づきあなたがその者に付与した特許ライセンスは終了します。

## 4. 表明

1. あなたには、上記のライセンスを付与する法的な権限があります。
2. 各 Contribution は、あなた自身のオリジナルな創作物であるか、または本 CLA の下でそれを提出する
   権限をあなたが有しています。
3. あなたの Contribution は、いかなる種類の保証もなく、**"AS IS"** の状態で提供されます。
4. 上記のいずれかの表明が不正確なものとなった場合、あなたは Project に通知します。

## 5. その他

本契約は、アメリカ合衆国の **カリフォルニア州** 法（その抵触法規定を除く）により規律されます。
いずれかの条項が執行不能と判断された場合でも、残りの条項は引き続き効力を有します。
