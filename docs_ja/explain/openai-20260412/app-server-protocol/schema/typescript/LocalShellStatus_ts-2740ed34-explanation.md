    // LocalShellStatus の 3 値のみを許可
    return value === "completed" ||
           value === "in_progress" ||
           value === "incomplete";

    status = externalStatus;                   // OK: 型ガードにより LocalShellStatus と確定

- このように **型ガード関数** を用意すると、コンパイル時の型安全性と実行時の安全性の両方を確保できます。
